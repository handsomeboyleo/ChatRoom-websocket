# wave-chat 在线聊天室

## 开源项目，前端使用 vue 框架开发页面，后端使用 websocket+node 完成后台的逻辑处理，项目在不断完善，欢迎联系交流！

## Project setup

```
yarn install
```

### Compiles and hot-reloads for development

```
yarn start
```

### Compiles and minifies for production

```
yarn run build
```

### Lints and fixes files

```
yarn run lint
```
